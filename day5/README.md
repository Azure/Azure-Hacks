# Day 5 - Architecting on Azure: Foundations

## Goal 🎯

This module kicks off a two-day architecture hackathon where you will design and implement a secure, scalable N-tier application on Microsoft Azure. Day 5 focuses on the foundational pillars — Compute, Storage, and Monitoring — building upon the security and infrastructure knowledge from previous days.

By the end of this day you will have:
- Understood the three pillars of cloud architecture: Compute, Storage, Connectivity
- Selected an architecture blueprint for an N-tier application
- Deployed a React Web App + Python API + MongoDB stack on Azure
- Monitored your deployed resources with Azure Monitor
- Created an initial architecture diagram of your solution

## Prerequisites

- An Azure subscription (the same one you have been using since Day 1)
- Tools installed during [Day 1 - Chapter 0](../day1/chapter-0/README.md): Azure CLI, Terraform
- Completion of previous days' content

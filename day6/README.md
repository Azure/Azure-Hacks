# Day 6 - Architecting on Azure: Networking & Security

## Goal 🎯

Day 6 continues the architecture hackathon with a focus on networking and security. You will organize your resources into virtual networks, establish private connectivity, secure your application, and apply the Azure Well-Architected Framework to validate your design.

By the end of this day you will have:
- Organized Azure resources within a Virtual Network (VNet)
- Established Private Link connectivity between services
- Secured resources with Network Security Groups and access restrictions
- Configured RBAC permissions using Azure Key Vault
- Reverse-engineered a complete reference architecture
- Applied the Azure Well-Architected Framework to assess your design

## Prerequisites

- An Azure subscription (the same one you have been using since Day 1)
- Completion of [Day 5 - Architecting on Azure: Foundations](../day5/README.md)
